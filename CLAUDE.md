## Project

**Name:** LifeOS

**Goal:**

Build a personal operating system that centralizes my daily workflow into a single platform.

LifeOS is not a productivity app.
It is a modular personal ecosystem containing multiple integrated applications that I personally use every day.

Current planned modules:

- Dashboard
- Cyber Lab
- Trading Terminal
- Genshin Hub
- AI Workspace

The project prioritizes learning, long-term maintainability, and real-world usefulness over rapid feature development.
